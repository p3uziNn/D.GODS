# 🎮 D. GODS — Official Website

Website oficial da organização **D. GODS**, focada em e-sports, criação de conteúdo e construção de comunidade digital.

Projeto desenvolvido com foco em identidade visual forte, presença online e estruturação de marca.

---

## 🚀 Sobre a D. GODS

A **D. GODS** é uma organização voltada para:

- 🎮 E-sports
- 🎥 Criação de conteúdo (gameplay, shorts, lives)
- 🎤 Cultura digital (rap, anime, lifestyle)
- 🌐 Construção de comunidade via Discord

O site foi criado para fortalecer a presença digital da organização e centralizar informações da equipe, membros e projetos.

---

## 📌 Objetivos do Projeto

- Construir presença online profissional
- Apresentar membros da organização
- Centralizar links e redes sociais
- Reforçar identidade visual da marca
- Servir como base para expansão futura

---

## 🛠 Tecnologias Utilizadas

- HTML5 (estrutura semântica)
- CSS3
- JavaScript
- Design responsivo

## 🎨 Foco em Identidade Visual

O projeto foi desenvolvido com foco em:
